Development Environemnt

1) Linux Setup

There are two approaches to establish and access Linux environment from your computer. Approach #1 listed in this section is preferred for it allows you to use the Linux with super user privilege.

1a) Approach #1: Linux Development Server (ADS or LDS)

  • Download a 32bits Ubuntu Linux OS image from Ubuntu Download.
  • Download VMWare Player from VMWare Download and install it on your computer.
  • Follow instructions to install Ubuntu as guest OS on VMWare Player: Install Ubuntu

1b) Approach #2: Linux Guest OS on VMWare

Refer to Smart Agent Development Environment Page for information about requesting ADS account and using VNC client to connect to ADS server.

2) Workspace Setup

  • Download Maven 3 and install it on Linux.
  • Download JRuby 1.7.9 and install it on Linux.
  • Download jrubycForMaven Plugin and run "mvn clean install" to install the plugin in your local Maven repository.
  • Pull the NESAL source code from Git repository. (TBD)
  • Run the following commands to start a build of NESAL package.
cd nesal
mvn install

3) IDE Setup

  • Install Eclipse for Java EE
  • Import Maven project in Eclipse

4) Additional Notes for Ruby

  • Ruby source code should be placed in nesal/src/main/ruby directory and will be compiled to Java source code after running "mvn compile".
